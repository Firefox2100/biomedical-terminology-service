# Experiments

This directory contains the code for experiments around this project. They are not meant to be run as part of the main project, but rather as standalone scripts for testing and experimentation purposes. They are included here for convenience and to keep them organized in one place. They may be used for testing new features, debugging, or simply for learning and exploration.

Do not use these scripts in production, or even with production database, because they may be designed for an outdated database version, may unintentionally delete things from the database, and they are definitely not tested. Most are designed for HPC, so running on a normal machine will quickly drain all available resources. If you plan to reproduce any of the experiments, make sure to read the code and understand what it does before running it.
