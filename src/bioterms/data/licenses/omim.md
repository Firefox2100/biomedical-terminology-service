# Licence of Online Mendelian Inheritance in Man (OMIM) data

This project utilises the data from the Online Mendelian Inheritance in Man (OMIM) project, provided by the UMLS. However, this project is in no way affiliated with or endorsed by the UMLS.

The project takes the original data release, selects a subset of data it needs, and represent the data in a different format. The project does not modify the original data in any way, and the original data is provided as-is. If any discrepancies are found in the data, please refer to the original data release, and contact us to report an issue or error in our usage of it.

## Original license terms

This ontology is made available via the UMLS. Users of all UMLS ontologies must abide by the terms of the [UMLS license](https://uts.nlm.nih.gov/uts/assets/LicenseAgreement.pdf).
