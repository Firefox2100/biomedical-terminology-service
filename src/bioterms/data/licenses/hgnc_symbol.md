# Licence of HUGO Gene Nomenclature Committee (HGNC) data

This project utilises the data from the HUGO Gene Nomenclature Committee (HGNC) standard, provided by the HGNC. However, this project is in no way affiliated with or endorsed by the HGNC.

The project takes the original data release, selects a subset of data it needs, and represent the data in a different format. The project does not modify the original data in any way, and the original data is provided as-is. If any discrepancies are found in the data, please refer to the original data release, and contact us to report an issue or error in our usage of it.

## Original license terms

The HUGO Gene Nomenclature Committee (HGNC) wants to ensure that our information is accessible to the public and free to use, for this reason all data is released under the Creative Commons Public Domain (CC0) License. This means that any form of reuse of the content is permitted. For further information, please visit the CC0 page or the Creative Commons FAQ about this type of license.
